# Thinking Hats — Mode Catalog

This catalog extends the core hat system with reusable ways to apply the hats.

## 1. Analytic Pass
**Use when:** You want a disciplined examination of a decision, plan, or claim.

**Suggested flow:** Blue → White → Yellow → Black → Blue

**What it does:** Separates facts, upside, and risk before synthesis.

## 2. Socratic Interrogation by Hat Lens
**Use when:** You want better questions instead of immediate answers.

Each hat becomes an interrogator:
- **Blue:** What are we really trying to decide? What structure would help?
- **White:** What do we know, what do we think we know, and what is missing?
- **Red:** What feels charged, off, exciting, or uneasy here?
- **Yellow:** What value might be hidden here? What works in its favor?
- **Black:** What assumption is most fragile? What fails first?
- **Green:** What else could this be? What alternatives are we not seeing?

**Suggested flow:** Blue → selected hat lenses → Blue

## 3. Decision Comparison
**Use when:** You are choosing between multiple options.

Run the SAME hat order across each option so comparison is fair.

**Suggested flow:** Blue → White compare → Yellow compare → Black compare → Red pulse → Blue

## 4. Pre-Mortem
**Use when:** A plan looks exciting but may hide failure modes.

**Suggested flow:** Blue → White → Black → Green → Yellow → Blue

Black assumes the project failed, then Green and Yellow recover stronger alternatives.

## 5. Post-Mortem / Retrospective
**Use when:** Something already happened and you want useful learning.

**Suggested flow:** Blue → Red → White → Yellow → Black → Green → Blue

This sequence captures feeling first, then facts, then value, then failure, then improvement.

## 6. Rapid Pulse Check
**Use when:** You need a fast answer in under 5 minutes.

**Suggested flow options:**
- Blue → Yellow → Black → Blue
- Blue → Red → White → Blue
- Single-hat direct use

## 7. Creative Provocation Cycle
**Use when:** The team is stuck in obvious answers.

**Suggested flow:** Blue → Green → Green → Black → Green → Blue

The middle Black pass is deliberately brief so it sharpens, not kills, the idea stream.

## 8. Tension Scan
**Use when:** You suspect the issue is psychologically or politically loaded.

**Suggested flow:** Blue → Red → White → Black → Yellow → Blue

This exposes hidden discomfort without letting it dominate the whole session.

## 9. Recovery Sequence
**Use when:** A session is muddled because too many hats got mixed.

**Suggested flow:** Blue reset → White → one chosen hat → Blue

Use this to simplify, not to add more complexity.

## Selection Rule
Choose the lightest mode that still creates clarity. If a single direct hat pass will do, do not force a full sequence.
