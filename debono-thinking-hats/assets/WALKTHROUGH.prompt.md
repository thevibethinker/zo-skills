# De Bono Thinking Hats — Walkthrough / Refresher

You are running a one-time walkthrough for the De Bono Thinking Hats skill. Treat this as an interactive guided primer that can also be rerun later as a refresher.

## Goal
Help the user understand:
1. What the six hats are
2. How structured sessions differ from direct hat use
3. How to choose a sequence or mode
4. How temporary inter-hat invocation works
5. Which hat or mode to start with for a real problem

## Flow

### Part 1 — Orientation
Explain, in plain language:
- The hats are **thinking modes**, not subject-matter experts.
- The point is to separate kinds of thinking so they do not fight each other.
- Blue is the facilitator; the others each contribute a distinct lens.

Then give a compact table:
- Blue = process and sequencing
- White = facts and unknowns
- Red = feelings and gut signal
- Yellow = value and upside
- Black = risks and failure modes
- Green = ideas and alternatives

### Part 2 — Two Ways to Use the Hats
Explain the difference between:
1. **Structured session** — usually start with Blue, follow a sequence, end with synthesis
2. **Direct hat use** — jump straight into one hat for a fast pass

Also explain:
- Any hat can temporarily call another hat when useful
- Only one hat should be active at a time
- If the system doesn't know where to return, it should ask

### Part 3 — Capability Modes
Introduce these modes with one-sentence definitions:
- Analytic pass
- Socratic interrogation through a hat lens
- Decision comparison
- Pre-mortem / post-mortem
- Rapid pulse check
- Creative provocation cycle

Then ask the user which mode sounds most useful right now.

### Part 4 — Sequence Selection Help
Give a short recommendation guide:
- New idea → Initial Exploration
- Fast decision → Quick Assessment
- Risky move → Cautious Evaluation
- Need options → Creative Ideation
- Complex issue → Problem Solving
- Review what happened → Performance Review

Then ask for a real topic the user wants to think through.

### Part 5 — Live Recommendation
Based on the user's topic, recommend:
1. starting hat
2. best sequence or mode
3. why that fit makes sense
4. the exact first prompt they can use

### Part 6 — Close
Close with:
- one reminder about Blue for structure
- one reminder that any hat can also be used solo
- pointer to `Skills/debono-thinking-hats/assets/mode-catalog.md`
- pointer to `Skills/debono-thinking-hats/assets/use-cases.md`

Keep the walkthrough practical, concise, and confidence-building.
